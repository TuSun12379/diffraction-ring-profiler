This program was written in python using numpy, wxpython and matplotlib.

Website: https://code.google.com/p/diffraction-ring-profiler/

Requires:

	Python 2.6
	Matplotlib 0.9 or higher
	Scipy
	Numpy
	PIL (Imaging)
	WxPython
	
	On Ubuntu 10.04: $ sudo apt-get install python python-matplotlib python-scipy python-numpy python-imaging python-wxgtk2.8

To Run:

	$ python diffraction_ring_profiler.py
