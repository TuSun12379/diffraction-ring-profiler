This program was written in python using numpy, wxpython and matplotlib.

Website: https://code.google.com/p/diffraction-ring-profiler/

Tutorial: http://code.google.com/p/diffraction-ring-profiler/wiki/howtoscreenshots

Requires:

    Python 3.5+
    Matplotlib 2.0+ or higher
    Scipy
    Numpy
    PIL (Imaging)
    WxPython 4.0+

All these come with Anaconda Python 3.6 Distribution: https://www.continuum.io/downloads
conda install -c newville wxpython-phoenix

To Run:

    $ python diffraction_ring_profiler.py
